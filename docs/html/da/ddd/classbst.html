Detailed Description

template<typename kt, typename vt, typename cmp = std::less<kt>>
class bst< kt, vt, cmp >

Templated implementation of a binary search tree class.

Template Parameters
ktType of node keys.
vtType of node values.
cmpType of the custom comparison operator between keys. Default std::less<kt>.
